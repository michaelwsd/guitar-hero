# Guitar Hero
This is a project built using Typescript and RxJS, utilizing functional reactive programming principles. 

## Running the Code
Setup (requires node.js):

```bash
> npm install
```

Serve up the App (and ctrl-click the URL that appears in the console)

```bash
> npm run dev
```
